Chương trình "Hello, World!" là một truyền thống lâu đời trong lập trình. Nó là một chương trình đơn giản dùng để kiểm tra xem môi trường lập trình của bạn đã được cài đặt đúng cách hay chưa.

## Code

Trong Python, để in ra dòng chữ "Hello, World!", bạn chỉ cần sử dụng hàm `print()`.

```python
print("Hello, World!")
```

## Cách chạy chương trình

1.  Mở một trình soạn thảo văn bản (như VS Code, Sublime Text, hoặc Notepad).
2.  Gõ vào dòng code trên.
3.  Lưu file với tên `hello.py`.
4.  Mở terminal (hoặc Command Prompt) và di chuyển đến thư mục bạn đã lưu file.
5.  Gõ lệnh sau và nhấn Enter:

```bash
python hello.py
```

## Kết quả

Nếu mọi thứ đều đúng, bạn sẽ thấy kết quả sau được in ra màn hình:

```
Hello, World!
```

Vậy là bạn đã viết và chạy thành công chương trình Python đầu tiên của mình!
